Strudel Kochbuch

Ein Strudel-Lehrbuch in Strudel. Jedes Kapitel ist lauffähiger Code mit eingebauter Erklärung. Tippe ein Kapitel an, dann Strg+Enter oder das Play-Symbol.

  1. 00Lies mich zuerst
    Bedienung
  2. 01Hello Sound
    Eine Note. Ein Sample.
  3. 02Mini-Notation
    Klammern, Sterne, Tilden
  4. 03Polyrhythmik
    stack, mehrere Spuren
  5. 04Euklidische Rhythmen
    Tresillo, Bossa, Aksak
  6. 05Skalen und Melodie
    note(), scale
  7. 06Akkorde und Voicings
    Akkorde
  8. 07Effekte
    Filter, Hall, Delay, Distortion
  9. 08Signale und Modulation
    sine, perlin, LFO
  10. 09Time Modifier
    fast, slow, every, mask
  11. 10Eigene Samples
    WAVs laden, slicen
  12. 11Song-Struktur
    arrange, Sektionen
  13. 12Mini-Track
    Alles zusammen
  14. 13Synthese tief
    Sound-Design aus Bordmitteln
  15. 14MIDI und IO
    Hardware-Knobs, externe Synths
  16. 15Performance-Hygiene
    Live-Set-Tipps
  17. 16Eigene Helper
    register(), Achsen
  18. 17Hap-Internals
    Pattern-Theorie
  19. 18Genre-Kochbuch
    Vorlagen pro Genre
  20. 19Valenz und Arousal
    Stimmungs-Achsen
  21. 20Cheatsheet
    Spickzettel

Hinweise

  • Funktioniert in Chrome und Safari (auch iPad).
  • Erstes Sample-Trigger kann 1-2 Sekunden Stille sein — Sample wird geladen.
  • Master-Volume rechts unten im REPL.
  • Vollständiger Strudel-Editor mit allen Features: strudel.cc
